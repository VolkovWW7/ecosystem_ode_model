# Программный комплекс математического моделирования и калибровки процессов биоокисления (BioOx Simulation)

Данное программное обеспечение предназначено для моделирования динамики микробиологических систем в биореакторе. В основе комплекса лежит жесткая система обыкновенных дифференциальных уравнений (ОДУ), описывающая взаимодействие популяций автотрофов (водорослей *Chlorella*) и гетеротрофов (бактерий), а также динамику субстратов, детрита, БЖУ и энергетических пулов (АТФ).

Программа оснащена кроссплатформенным графическим интерфейсом и модулем автоматического подбора (калибровки) кинетических параметров на основе генетических алгоритмов.

## Основные возможности

* **Численное интегрирование систем ОДУ:** Использование современного адаптивного метода `LSODA` (из библиотеки `SciPy`) для решения жестких биологических систем.
* **Интерактивный GUI:** Удобный интерфейс на базе **PySide6 (Qt)**, позволяющий на лету изменять начальные условия, параметры среды и константы.
* **Встроенная научная визуализация:** Графики популяций, детрита, БЖУ и АТФ интегрированы непосредственно в окна вкладок интерфейса благодаря связке `Matplotlib` и Qt.
* **Эволюционная калибровка:** Модуль автоматической оптимизации параметров на базе алгоритма дифференциальной эволюции (`differential_evolution`), минимизирующий среднеквадратичную относительную ошибку (RMSRE) по сравнению с лабораторными данными.
* **Многопоточность:** Тяжелые вычисления калибровки вынесены в отдельный фоновый поток (`QThread`), что сохраняет интерфейс программы отзывчивым во время расчетов.
* **Управление конфигурациями:** Импорт и экспорт настроек и результатов калибровки в универсальном формате JSON.

## Архитектура проекта

* `main.py` — Главная точка входа, реализующая паттерн MVC (контроллер), управляющая потоками данных и жизненным циклом приложения.
* `gui.py` — Модуль графического интерфейса. Описывает структуру главного окна, логику виджетов, вкладок и Canvas для графиков.
* `mathmodel.py` — Математическое ядро модели. Содержит систему ОДУ, дефолтные параметры, функции расчета и генерации графиков.
* `calibrate_gui.py` — Модуль оптимизации. Настраивает границы поиска параметров, вычисляет целевую функцию RMSRE и запускает генетический алгоритм.

## Требования к системе и окружение

Для запуска исходного кода требуется **Python 3.10+**.

Основные зависимости:
* `PySide6` — Графический интерфейс Qt.
* `numpy` — Векторно-матричные вычисления.
* `scipy` — Решение ОДУ (`solve_ivp`) и оптимизация (`differential_evolution`).
* `matplotlib` — Построение графиков и холсты интеграции в Qt.

## Установка и запуск

1. Клонируйте репозиторий или перейдите в папку с проектом:
   ```bash
   cd bioox-simulation
   ```
2. Создайте виртуальное окружение
    ```bash
    # Windows
    python -m venv venv
    venv\Scripts\activate

    # Linux / macOS
    python3 -m venv venv
    source venv/bin/activate
    ```
3. Установите зависимости
    ```bash
    pip install pyside6 numpy scipy matplotlib
    ```
4. Запуск приложения
    ```bash
    python3 main.py
    ```
